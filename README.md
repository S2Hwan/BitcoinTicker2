# BitcoinTicker2
37가지 가상화폐의 시세를 실시간으로 확인할수 있는 iOS Application 

## Features
 - [x] 실시간 비트코인 시세정보 확인
 - [x] 비트코인, 이더리움 등 37가지 가상화폐 정보 제공
